# rwqe-dev.github.io
RWQE website

> **⚙️How publishing works**
> If you are wondering how changes get to the live website, we use a simple automated system:
> * **The `online` branch:** In GitHub, we keep the live website files in a special workspace called the `online` branch. 
> * **The Automation:** Whenever someone saves a new update to this `online` branch, GitHub takes over. An automatic script (a GitHub Action) builds the website and publishes it to the web for us. No extra software or manual uploading is required!
